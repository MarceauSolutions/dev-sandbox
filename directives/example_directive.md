# Directive: Example Task

## Goal
This is an example directive template showing the standard structure. Replace this with your actual task description.

## Inputs
- **Input 1**: Description of the first required input
- **Input 2**: Description of the second required input
- **Optional Input**: Description of optional parameters

## Tools
- `execution/example_script.py` - Brief description of what this script does
- `execution/another_script.py` - Description of another tool if needed

## Process
1. **Validate inputs**: Check that all required inputs are provided and valid
2. **Execute main task**: Run the primary execution script with inputs
3. **Handle results**: Process the output from the script
4. **Generate deliverable**: Create the final output (Google Sheet, etc.)
5. **Notify if needed**: Send notifications about completion

## Outputs
- **Primary Output**: Description of main deliverable (e.g., "Google Sheet with processed data")
- **Secondary Output**: Any additional outputs or side effects

## Edge Cases
- **Missing inputs**: How to handle when required inputs are missing
- **API failures**: How to retry or recover from API errors
- **Rate limits**: How to handle rate limiting (wait, batch, etc.)
- **Invalid data**: How to handle malformed or unexpected input data
- **Partial success**: What to do when some items succeed and others fail
