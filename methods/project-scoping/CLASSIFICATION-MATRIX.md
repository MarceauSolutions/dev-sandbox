## Quick Classification Guide

### Step 1: Determine Project Tier

| If the project has... | It's likely... |
|----------------------|----------------|
| 1-3 pages, no CMS, basic form | **Starter** |
| 4-7 pages, simple CMS, contact forms | **Standard** |
| 8-15 pages, full CMS, booking/integrations | **Professional** |
| 15+ pages, custom features, API work | **Enterprise** |

### Step 2: Check for Complexity Multipliers

| If the project includes... | Apply... |
|---------------------------|----------|
| E-commerce/shopping cart | 1.5-2x |
| User accounts/membership | 1.3x |
| Each third-party API | 1.2x |
| Complex animations | 1.2x |
| AI/automation features | 1.5-2x |
| Rush delivery (<2 weeks) | 1.5x |
| Content creation needed | 1.2-1.5x |

### Step 3: Calculate Price

```
Base Price (from tier) × Multipliers = Final Price
```

## Tier Definitions

### Tier 1: Starter ($1,500-2,500)

**Description**: Simple online presence for new or budget-conscious businesses

**Scope Includes**:
- 1-3 pages (Home, About, Contact)
- Template-based design (WordPress, Squarespace, Webflow)
- Mobile responsive
- Basic contact form
- Google Analytics setup
- Basic SEO (meta tags, sitemap)

**Scope Excludes**:
- Content writing (client provides)
- Photography (client provides or stock)
- CMS for client editing
- Ongoing maintenance
- Additional pages

**Timeline**: 1-2 weeks

**Target Client**: New businesses, side hustles, event pages, "I just need something online"

**Payment**: 50/50 (start/completion)

---

### Tier 2: Standard ($3,500-5,500)

**Description**: Professional website for established small businesses

**Scope Includes**:
- 4-7 pages
- Custom design within template framework
- Simple CMS (client can update content)
- Contact forms with email notifications
- Social media integration
- Full SEO optimization
- Google Business Profile optimization
- 1 month post-launch support

**Scope Excludes**:
- E-commerce functionality
- User accounts/membership
- Complex integrations
- Content writing (unless added)
- Ongoing maintenance

**Timeline**: 4-6 weeks

**Target Client**: Gyms, fitness studios, local services, restaurants, salons

**Payment**: 50/25/25 (start/design approval/launch)

---

### Tier 3: Professional ($6,000-10,000)

**Description**: Full-featured website for growing businesses

**Scope Includes**:
- 8-15 pages
- Fully custom design
- Advanced CMS with content types
- Booking/scheduling integration (Mindbody, Acuity, etc.)
- Email marketing integration
- Advanced forms with conditional logic
- Performance optimization
- Content strategy consultation
- 3 months post-launch support

**Scope Excludes**:
- E-commerce (or add multiplier)
- Custom web application features
- API development
- Ongoing SEO/marketing

**Timeline**: 6-10 weeks

**Target Client**: Multi-location businesses, franchises, professional services, growing companies

**Payment**: 50/25/25 or 40/30/30 for larger projects

---

### Tier 4: Enterprise ($10,000+)

**Description**: Custom web platform or application

**Scope Includes**:
- 15+ pages or unlimited for CMS-driven
- Fully custom design and development
- Custom functionality
- Third-party API integrations
- User authentication/membership
- Custom database
- Advanced security
- Performance optimization
- Training and documentation
- 6 months support

**Scope Excludes**:
- Defined per project (everything else is change order)

**Timeline**: 10+ weeks

**Target Client**: Established businesses, startups needing MVP, franchise systems

**Payment**: 4+ milestone structure (30/25/25/20)

## Complexity Multipliers Detail

### E-commerce (1.5-2x)

| Complexity | Multiplier | Includes |
|------------|------------|----------|
| Simple (1-20 products, no variants) | 1.5x | Shopify/WooCommerce basic |
| Medium (20-100 products, variants) | 1.7x | Inventory management, shipping |
| Complex (100+ products, custom) | 2x+ | Custom checkout, subscriptions |

### Membership/Login (1.3x)

| Type | Multiplier | Examples |
|------|------------|----------|
| Simple gated content | 1.2x | Members-only page |
| User profiles | 1.3x | Dashboard, settings |
| Role-based access | 1.5x | Multiple membership levels |

### Third-Party Integrations (1.2x each)

| Integration Type | Complexity | Notes |
|-----------------|------------|-------|
| Well-documented API | 1.1-1.2x | Stripe, Mailchimp |
| Moderately documented | 1.2-1.3x | Most booking systems |
| Poorly documented/legacy | 1.4-1.5x | Add discovery phase |

### AI/Automation (1.5-2x)

| Type | Multiplier | Examples |
|------|------------|----------|
| Pre-built AI (ChatGPT widget) | 1.2x | Chatbot iframe |
| Custom chatbot | 1.5x | Trained on client content |
| Workflow automation | 1.3-1.5x | Zapier/Make integrations |
| Custom AI/ML | 2x+ | Always T&M with cap |

### Rush Delivery (1.5x)

| Standard Timeline | Rush Timeline | Premium |
|-------------------|---------------|---------|
| 2 weeks | <1 week | 1.5x |
| 4-6 weeks | <2 weeks | 1.5x |
| 8+ weeks | <4 weeks | 1.5x |

## Example Classifications

### Example 1: Local Gym Website

**Inquiry**: "5-page website with class schedule and booking integration"

**Classification**:
- Pages: 5 → Standard tier
- Booking integration: +1.2x multiplier
- Base: $4,500 × 1.2 = **$5,400**

**Quote**: $5,400 (Standard + booking integration)

---

### Example 2: Fitness Studio with Membership

**Inquiry**: "Website with members-only video library and Mindbody booking"

**Classification**:
- Pages: ~10 → Professional tier
- Membership: +1.3x
- Booking API: +1.2x
- Base: $7,500 × 1.3 × 1.2 = **$11,700**

**Quote**: $11,700 → Round to **$12,000** (move to Enterprise tier)

---

### Example 3: Simple Landing Page

**Inquiry**: "Just a one-page site for my personal training business"

**Classification**:
- Pages: 1 → Starter tier
- No multipliers
- Base: **$1,500-1,800**

**Quote**: $1,750 flat

---

### Example 4: E-commerce for Local Boutique

**Inquiry**: "Online store with 50 products, shipping, local pickup"

**Classification**:
- Pages: ~8-10 → Professional tier
- E-commerce (medium): +1.7x
- Base: $7,500 × 1.7 = **$12,750**

**Quote**: $12,750 → **$13,000** (Enterprise due to complexity)
