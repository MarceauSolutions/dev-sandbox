# Exploration Plan: Project Scoping Method

*Created: 2026-01-15*

## Objective

Research best practices for project classification, pricing, and estimation to create a robust scoping framework for Marceau Solutions.

## Agent Assignments

### Agent 1: Pricing Research
**Focus**: Market rates and pricing strategies

**Research Questions**:
1. What do web design agencies charge in Naples/SW Florida?
2. What's the national average for similar services?
3. Value-based pricing vs hourly: when to use each?
4. How to structure recurring revenue (maintenance contracts)?
5. What pricing models work best for solopreneurs?

**Deliverable**: `agent1-pricing/FINDINGS.md`

---

### Agent 2: Intake Process
**Focus**: Lead qualification and discovery

**Research Questions**:
1. What questions effectively qualify web design leads?
2. How to scope projects without over-promising?
3. What are red flags indicating difficult clients?
4. Discovery call best practices and scripts?
5. How to handle "I don't know what I want" clients?

**Deliverable**: `agent2-intake/FINDINGS.md`

---

### Agent 3: Estimation Models
**Focus**: Project estimation and contracts

**Research Questions**:
1. Fixed price vs Time & Materials: decision framework?
2. How to handle scope creep contractually?
3. What contingency buffer percentages to use?
4. Milestone-based vs upfront payment structures?
5. How to estimate accurately for custom features?

**Deliverable**: `agent3-estimation/FINDINGS.md`

---

### Agent 4: Tooling & Automation
**Focus**: Tools and automation for scoping

**Research Questions**:
1. Can intake forms auto-generate estimates?
2. Best CRM for tracking project pipelines?
3. Proposal template tools (PandaDoc, Proposify, etc.)?
4. Contract/SOW generators for freelancers?
5. How to integrate scoping with existing ClickUp workflow?

**Deliverable**: `agent4-tooling/FINDINGS.md`

---

## Evaluation Criteria

Each agent should score their findings on:
- **Applicability**: How relevant to a solopreneur in Naples?
- **Actionability**: Can we implement this immediately?
- **Evidence**: Is this based on data or opinion?
- **Risk**: What could go wrong with this approach?

## Consolidation

After all agents complete, consolidate into:
1. `consolidated/COMPARISON-MATRIX.md` - Side-by-side findings
2. `consolidated/RECOMMENDATION.md` - Final framework

## Timeline

- Agent research: 1-2 hours (parallel)
- Consolidation: 30 minutes
- Total: ~2.5 hours
