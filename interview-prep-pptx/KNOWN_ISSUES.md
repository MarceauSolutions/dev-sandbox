## Potential Frontend Issues to Watch

These are areas that may need attention based on the implementation:

1. **Large file uploads** - Resume uploads may timeout on slow connections
2. **Mobile responsiveness** - Slide preview grid may need adjustment on small screens
3. **Download failures** - PPTX download may fail if generation takes too long
4. **Session persistence** - Browser refresh loses current session state
5. **Image generation** - Grok API rate limits could cause failures on rapid regeneration

## How to Report Issues

When you encounter an issue:

1. Add it to the appropriate section above
2. Include:
   - Brief description
   - Steps to reproduce (if applicable)
   - Browser/environment details
   - Screenshots if helpful

Example:
```markdown
| Slide preview not loading | Open | Chrome 120, macOS - images fail to render after generation |
```
