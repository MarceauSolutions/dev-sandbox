# Claude Development Framework

A complete "operating system" for AI-assisted development with Claude Code.

## What This Is

A structured framework that transforms Claude from a helpful assistant into an autonomous development partner. Built over months of real-world development work, this system includes:

- **Claude.md** - Master configuration file with communication patterns
- **17 SOPs** - Standard Operating Procedures covering the full development lifecycle
- **DOE Architecture** - Directive-Orchestration-Execution pattern for reliable AI development
- **Multi-Agent Workflows** - Parallel development and testing with specialized agents
- **Living Documentation** - Patterns that evolve and improve with each session

## The Problem This Solves

Using Claude without structure leads to:
- Inconsistent results session to session
- Lost context and learnings
- Reinventing the wheel every time
- No memory of what worked before
- Difficulty scaling development

## Who This Is For

- **Solo entrepreneurs** building products with AI

## Quick Start

1. Copy `claude.md` to your project root
2. Read the "Session Start Checklist" section
3. Start a Claude Code session
4. Claude will automatically follow the patterns

## Core Concepts

### DOE Architecture
```
Layer 1: DIRECTIVE (what to do)
    ↓
Layer 2: ORCHESTRATION (Claude coordinates)
    ↓
Layer 3: EXECUTION (scripts do the work)
```
