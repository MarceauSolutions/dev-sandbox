MY TASKS:
1. Database Schema (PostgreSQL)
   ```sql
   CREATE TABLE mcps (
       id UUID PRIMARY KEY,
       name VARCHAR(255) UNIQUE NOT NULL,
       provider VARCHAR(255) NOT NULL,
       category VARCHAR(100) NOT NULL,
       capabilities JSONB NOT NULL,
       endpoint URL,
       pricing DECIMAL(10,4) NOT NULL,
       sla_uptime DECIMAL(5,2) NOT NULL,
       rating DECIMAL(3,2),
       total_requests INTEGER DEFAULT 0,
       status VARCHAR(50) DEFAULT 'active',
       created_at TIMESTAMP DEFAULT NOW()
   );

   CREATE TABLE transactions (
       id UUID PRIMARY KEY,
       ai_agent_id VARCHAR(255) NOT NULL,
       mcp_id UUID REFERENCES mcps(id),
       request_type VARCHAR(100) NOT NULL,
       mcp_cost DECIMAL(10,4) NOT NULL,
       platform_fee DECIMAL(10,4) NOT NULL,
       latency_ms INTEGER NOT NULL,
       success BOOLEAN NOT NULL,
       created_at TIMESTAMP DEFAULT NOW()
   );
   ```
   Save to: database_schema.sql

2. Build MCP Registry (registry.py)
   - register_mcp(metadata): Add new MCP to database
   - find_mcps(intent, filters): Find MCPs that can handle request
   - get_health_status(mcp_id): Check uptime, latency, error rate
   - update_rating(mcp_id, rating): Update quality score

3. Build Routing Engine (router.py)
   - route_request(query, context): Main routing logic
   - score_and_select(mcps, context): Pick best MCP
   - Scoring algorithm:
     ```python
     score = (reliability * 0.40) +
             (latency * 0.30) +
             (cost * 0.20) +
             (user_preference * 0.10)
     ```
   - execute_mcp(mcp, query): Call selected MCP
   - Fallback logic: If primary fails, try 2nd best
   - Circuit breaker: Stop routing to failing MCPs

4. Build Billing System (billing.py)
   - log_transaction(ai_agent, mcp, cost): Record every request
