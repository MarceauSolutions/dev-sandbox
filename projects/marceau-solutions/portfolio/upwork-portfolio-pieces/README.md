## Portfolio Items

### 1. FBA Analytics Dashboard
**Location:** `fba-dashboard/`

Interactive dashboard showing:
- Revenue & profit trends (Chart.js)
- KPI cards (revenue, units, margin, ACOS)
- Top products table
- Restock alerts with urgency levels
- PPC campaign performance

**Screenshot:** Open `fba-dashboard/index.html` in browser, take full-page screenshot.

> Custom FBA analytics dashboard built with Python + SP-API. Real-time sync of sales, inventory, and advertising data. Automated alerts for low stock and high ACOS campaigns.

## How to Take Screenshots

1. **Open in Browser:**
   ```bash
   open fba-dashboard/index.html
   open restock-calculator/index.html
   ```

2. **Full Page Screenshot (Chrome):**
   - Open DevTools (Cmd+Option+I)
   - Cmd+Shift+P → "Capture full size screenshot"

3. **Or use a tool like CleanShot X:**
   - Capture scrolling screenshot
   - Add device frame if desired
