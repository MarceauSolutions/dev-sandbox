# Web Development & Automation Portfolio

A collection of modern web applications showcasing frontend development, business automation, and data visualization skills.

## Projects

### 1. TechFlow Landing Page
**`landing-page-demo/`**

A modern SaaS landing page with smooth animations and responsive design.

**Features:**
- Hero section with gradient effects and floating elements
- Feature grid with hover animations
- Pricing toggle (monthly/yearly) with live price updates
- Testimonial carousel
- Mobile-responsive navigation
- Scroll-triggered animations using Intersection Observer

**Tech Stack:** HTML5, CSS3 (CSS Variables, Flexbox, Grid), Vanilla JavaScript

**Live Preview:** Open `landing-page-demo/index.html` in browser

---

### 2. Business Analytics Dashboard
**`automation-dashboard/`**

A comprehensive analytics dashboard for business operations monitoring.

**Features:**
- Real-time revenue and traffic charts (Chart.js)
- KPI stat cards with trend indicators
- Recent orders table with status badges
- Top products listing
- Activity feed with timestamps
- Tab navigation for different time periods
- Animated number counters on load

**Tech Stack:** HTML5, CSS3, Vanilla JavaScript, Chart.js

**Live Preview:** Open `automation-dashboard/index.html` in browser

---

### 3. LeadHunter Pro - Lead Scraper
**`lead-scraper-showcase/`**

A dark-themed lead generation platform interface demonstrating B2B automation capabilities.

**Features:**
- Multi-source lead scraping (Google Places, Yelp, Apollo.io, LinkedIn)
- Lead qualification with pain point tagging (No Website, Few Reviews, No Social)
- Lead scoring system with visual indicators
- Real-time scraping progress with animated stats
- CSV export functionality
- Campaign management integration
- Responsive sidebar navigation

**Tech Stack:** HTML5, CSS3 (Dark Theme, CSS Variables), Vanilla JavaScript

**Live Preview:** Open `lead-scraper-showcase/index.html` in browser

## Design Philosophy

1. **Performance First** - Vanilla JS, no framework overhead
2. **Responsive** - Mobile, tablet, and desktop optimized
3. **Accessible** - Semantic HTML, proper contrast ratios
4. **Maintainable** - CSS variables for theming, modular JavaScript
