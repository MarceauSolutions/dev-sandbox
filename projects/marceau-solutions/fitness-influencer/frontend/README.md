# Fitness Influencer Assistant - Frontend

Web interface for the Fitness Influencer AI Assistant.

## Related Repositories

- **Frontend (this repo):** User interface (HTML/CSS/JS)
- **Backend:** [fitness-influencer-backend](https://github.com/MarceauSolutions/fitness-influencer-backend) - Python FastAPI server

## Deployment

- **Frontend:** GitHub Pages
- **Backend API:** Railway - `https://web-production-44ade.up.railway.app`

## Features

- 🎬 Video editing interface
- 🎨 Educational graphics creator
- 📧 Email digest viewer
- 📊 Revenue analytics dashboard
- 🤖 AI image generator
- 📅 Calendar management

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API for backend communication

## Local Development

```bash
# Simple HTTP server
python -m http.server 8080
# or
npx serve
```

Visit: http://localhost:8080
