# Workflow: AI News Digest Generation

**Created:** 2026-01-19
**Purpose:** Generate automated AI & job market news digests to stay informed about industry trends
**Output:** Markdown/JSON report with key developments, job market trends, skills in demand, and actionable insights

---

## Overview

This workflow uses web search capabilities to research and summarize the latest AI news, job market trends, and industry developments. It produces a structured digest that helps track AI's impact on work and identify emerging opportunities.

---

## Use Cases

- Daily morning digest of AI news (past 24 hours)
- Weekly deep dive into AI industry developments
- Track AI job displacement and hiring trends
- Monitor skills in demand for career planning
- Identify AI automation impacts on specific industries
- Research for client proposals or market positioning

---

## Prerequisites

Before generating digest:
- ✅ `ANTHROPIC_API_KEY` set in `.env` (for web search via Claude)
- ✅ Internet connection for web searches
- ✅ Output directory exists: `projects/personal-assistant/output/ai-digests/`

---

## Input Parameters

| Parameter | Options | Default | Description |
|-----------|---------|---------|-------------|
| `--weekly` | flag | False | Generate weekly deep dive instead of daily |
| `--prompt-only` | flag | False | Just output the prompt (don't generate) |
| `--template` | flag | False | Output blank template for manual filling |
| `--output` | json/markdown/both | both | Output format |

## Steps

### Step 1: Decide Digest Type

**Objective:** Choose daily (24 hours) or weekly (7 days) digest

**Actions:**
- Daily digest: Quick overview of yesterday's key developments (5-10 minutes to generate)
- Weekly digest: Deep dive with policy updates, funding news, statistics (20-30 minutes)

**Decision:**
- Use daily for morning routine integration
- Use weekly for weekend planning or strategy sessions

## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| Web search returns no results | Date filter too restrictive | Expand timeframe from 24h to 48h |
| Duplicate news items | Same story from multiple sources | Deduplicate by checking headline similarity |
| Missing source URLs | Claude didn't include links | Add explicit instruction: "Include source URL for every claim" |
| Low-quality sources | Not filtering for credible publishers | Add filter: "Only use sources from major tech news sites" |
| Too much noise | Query too broad | Narrow with specific keywords: "AI enterprise adoption" not just "AI news" |
| Outdated information | Claude's knowledge cutoff | ALWAYS use web search, never rely on Claude's training data |
