# Workflow: Client Onboarding & Setup

## Overview

This workflow documents the complete process for onboarding a new Elder Tech Concierge client, from initial consultation through successful handoff to ongoing support.

## Use Cases

- New customer signup (Essential or Premium tier)
- Beta customer onboarding
- Family-initiated setup for senior parent

## Prerequisites

- Client has agreed to service (verbal or signed)
- Payment processed (setup fee + first month)
- iPad and accessories available
- Technician calendar blocked (2-4 hours depending on tier)

## Equipment Checklist

### To Bring to Setup

- [ ] iPad (pre-configured with app)
- [ ] iPad case (senior-friendly, with stand)
- [ ] Charging cable and power adapter
- [ ] Screen protector (pre-applied)
- [ ] Quick reference card (laminated)
- [ ] Consent forms
- [ ] Contact information sheet
- [ ] Backup stylus (large grip)
- [ ] Portable speaker (for hearing-impaired clients)

### Software Pre-Configuration

- [ ] Elder Tech Concierge app installed
- [ ] App logged into client account
- [ ] Guided Access enabled (kiosk mode)
- [ ] Accessibility settings configured (large text, high contrast)
- [ ] Do Not Disturb exceptions set

## Steps

### Step 1: Pre-Visit Preparation (Day Before)

**Objective:** Gather all information needed for smooth setup

**Actions:**
1. **Confirm appointment** via phone call
   - Verify address and parking
   - Confirm who will be present (senior, family member, or both)
   - Ask about senior's primary concerns/goals
   - Note any physical limitations (hearing, vision, mobility)

2. **Prepare client account**
   - Create user account in system
   - Configure basic settings based on call
   - Pre-load any contacts already provided

3. **Prepare iPad**
   - Charge to 100%
   - Verify app is updated
   - Login to client account
   - Test voice interface

4. **Print materials**
   - Consent form (2 copies)
   - Quick reference card
   - Contact information sheet

**Tools:**
- Phone
- Admin dashboard
- iPad setup station

**Time:** 30 minutes

---

### Step 2: Initial Consultation (On-Site)

**Objective:** Build rapport and understand specific needs

**Actions:**
1. **Arrival & introductions** (10 minutes)
   - Introduce yourself and the service
   - Express that you're here to help, not judge tech ability
   - Identify the best location for the tablet (good lighting, accessible)

2. **Needs assessment** (20 minutes)
   - Ask: "What would you most like help with?"
   - Ask: "Who do you want to stay in touch with most?"
   - Ask: "What's been frustrating about technology?"
   - Ask: "Are there any health reminders you need?"
   - Listen more than talk

3. **Tour the home setup area**
   - Identify power outlet locations
   - Check WiFi signal strength
   - Note lighting conditions
   - Identify any accessibility needs (wheelchair, limited mobility)

4. **Review consent and pricing**
   - Explain what the service does and doesn't do
   - Review privacy policy (in plain English)
   - Get signatures on consent form
   - Confirm ongoing subscription

**Tools:**
- Consent forms
- Notepad for notes
- WiFi signal tester (phone app)

**Time:** 30-45 minutes

**Key Questions to Ask:**
```
1. "Who are the most important people you want to stay connected with?"
2. "Do you take any medications that need reminders?"
3. "How often do you check your email?"
4. "Do you have any upcoming doctor appointments?"
5. "Is there a family member who can help if you have questions?"
```

---

### Step 3: Hardware Setup

**Objective:** Install and configure the physical device

**Actions:**
1. **Position the iPad**
   - Place in chosen location
   - Ensure charging cable can reach outlet
   - Verify comfortable viewing angle
   - Test that senior can reach it comfortably

2. **Configure WiFi**
   - Connect to home WiFi network
   - Note password in secure client file
   - Test connection speed

3. **Accessibility adjustments**
   - Adjust text size with senior present
   - Set volume level to comfortable
   - Configure voice speed (usually slower)
   - Enable high contrast if needed

4. **Test audio**
   - Play sample voice output
   - Adjust volume
   - Test that senior can hear clearly
   - If needed, connect external speaker

**Tools:**
- iPad
- Charging equipment
- External speaker (if needed)

**Time:** 15-20 minutes

---

### Step 4: Contact Import & Configuration

**Objective:** Populate the contact list with important people

**Actions:**
1. **Gather contact information**
   - Use contact information sheet
   - Get: Name, relationship, phone, email
   - Ask for nicknames: "What do you call your daughter?"

2. **Enter contacts into system**
   - Primary emergency contact (usually adult child)
   - Secondary emergency contact
   - Doctor's office
   - Close family members (5-10 contacts typical)
   - Friends (optional)

3. **Configure nicknames**
   - "my daughter" = Sarah
   - "my son" = Michael
   - "the doctor" = Dr. Smith
   - "my best friend" = Carol

4. **Set emergency contacts**
   - Mark 1-2 contacts as emergency
   - Confirm phone numbers are correct
   - Test emergency alert (with warning to family)

**Contact Entry Template:**
```
Name: Sarah Johnson
Relationship: Daughter
Phone: +1-555-123-4567
Email: sarah@email.com
Nicknames: "my daughter", "Sarah", "sweetie"
Emergency Contact: Yes (Primary)
```

**Tools:**
- Admin dashboard (contact management)
- Contact information sheet

**Time:** 20-30 minutes

---

### Step 5: Software Walkthrough & Training

**Objective:** Teach the senior how to use the system

**Actions:**
1. **Introduction to the home screen** (10 minutes)
   - Show the big buttons
   - Explain what each one does
   - Let them tap each one once

2. **Voice interface training** (20 minutes)
   - Demonstrate: "Hey Concierge, what time is it?"
   - Have senior try: "Hey Concierge, what's the weather?"
   - Practice: "Text my daughter" flow (with actual text)
   - Practice: "Read my emails"

3. **Emergency button training** (5 minutes)
   - Show the red emergency button
   - Explain what happens when pressed
   - Practice pressing (with alert disabled or family warned)
   - Show the "I'm okay" cancellation

4. **Reminder training** (10 minutes)
   - Set a test reminder together
   - Wait for it to fire (or set for 2 minutes)
   - Show how to acknowledge
   - Show how to snooze

5. **Common scenarios practice**
   - "Send a text to [daughter's name]"
   - "Do I have any emails?"
   - "Remind me to take my medicine tomorrow at 8am"
   - "What's on my calendar today?"

**Training Phrases to Practice:**
```
"Hey Concierge, [request]"
"Text my daughter"
"Call my son"
"Read my emails"
"What's the weather?"
"Remind me to..."
"What's on my calendar?"
```

**Time:** 45-60 minutes

**Tips:**
- Go SLOW - repeat as needed
- Let them do it, don't grab the device
- Celebrate small wins ("Great job!")
- Don't overwhelm - 3-4 features max in first session
- Promise to come back for more training if needed

---

### Step 6: Family Member Orientation

**Objective:** Ensure family understands system and can support senior

**Actions:**
1. **Explain family features** (if family present)
   - Daily "all is well" check notifications
   - Emergency alert system
   - How to add reminders remotely (future)
   - Weekly activity summaries

2. **Set expectations**
   - What the service does and doesn't do
   - When to call support vs. troubleshoot
   - That the senior is in control

3. **Provide family dashboard access** (Premium tier)
   - Create family account
   - Show dashboard features
   - Set notification preferences

4. **Exchange contact information**
   - Family contact for emergencies
   - Support phone/email
   - Expected response times

**If family not present:**
- Schedule brief phone call with family member
- Send email summary of setup
- Provide family with support contact info

**Time:** 15-20 minutes

---

### Step 7: Documentation & Handoff

**Objective:** Leave the client with everything they need

**Actions:**
1. **Provide quick reference card**
   - Laminated card with common commands
   - Support phone number prominently displayed
   - Emergency button instructions

2. **Review what to do if confused**
   - "Just ask the Concierge for help"
   - "Call me at [phone] anytime"
   - "Your [daughter] can also help"

3. **Schedule follow-up**
   - Phone check-in: 2 days after setup
   - Optional in-home follow-up: 1 week (Premium)
   - Regular check-in: Monthly (included)

4. **Leave behind materials**
   - Quick reference card
   - Signed consent form (their copy)
   - Support contact card
   - Extra charging cable (Premium)

5. **Final test**
   - Have senior independently:
     - Send a text message
     - Ask about the weather
     - Find the emergency button
   - Confirm everything works

**Quick Reference Card Content:**
```
+------------------------------------------+
|     ELDER TECH CONCIERGE                 |
|     Quick Reference Card                 |
+------------------------------------------+
|                                          |
|  SAY: "Hey Concierge..."                 |
|                                          |
|  - "Text my daughter"                    |
|  - "Read my emails"                      |
|  - "What's the weather?"                 |
|  - "Remind me to..."                     |
|  - "What's on my calendar?"              |
|                                          |
|  EMERGENCY: Tap the red button           |
|                                          |
|  NEED HELP?                              |
|  Call: (555) 123-4567                    |
|  Email: support@eldertechconcierge.com   |
|                                          |
+------------------------------------------+
```

**Time:** 15-20 minutes

---

### Step 8: Post-Setup Administration

**Objective:** Complete administrative tasks and prepare for support

**Actions:**
1. **Update client record**
   - Mark setup as complete
   - Add notes from consultation
   - Record any special needs or preferences
   - Upload signed consent form

2. **Send confirmation email**
   - To senior (if they use email)
   - To family member
   - Include: Support contacts, what to expect, quick reference

3. **Schedule follow-up tasks**
   - 2-day phone check-in
   - 1-week follow-up (Premium)
   - Add to monthly check-in rotation

4. **Request testimonial setup**
   - Note to ask for testimonial at 2-week mark
   - If positive experience, request referral

**Time:** 20-30 minutes (back at office)

## Troubleshooting Common Issues

### During Setup

| Issue | Solution |
|-------|----------|
| Weak WiFi signal | Try repositioning iPad, suggest WiFi extender |
| Senior can't hear voice | Increase volume, try external speaker |
| Voice recognition poor | Speak slower, reduce background noise, adjust mic |
| Senior frustrated | Take a break, simplify to 2 features, promise follow-up |
| Family member takes over | Gently redirect: "Let's let Mom try it herself" |

### Post-Setup

| Issue | Solution |
|-------|----------|
| "It's not listening" | Check WiFi, restart app, verify wake word |
| "I forgot how to..." | Refer to quick reference card, offer phone walkthrough |
| "My daughter isn't getting texts" | Verify phone number, check contact spelling |
| "The reminders are annoying" | Adjust timing or frequency, reduce to essential only |

## Success Criteria

### Immediate (End of Setup)

- [ ] Senior can independently send a text message
- [ ] Senior can independently ask about weather
- [ ] Senior knows where emergency button is
- [ ] All key contacts entered with nicknames
- [ ] At least 1 reminder set (medication or appointment)
- [ ] Family member has support contact info
- [ ] Consent form signed and collected

### Short-Term (2 Weeks)

- [ ] Senior using device at least 3x per week
- [ ] No major support issues
- [ ] Family reports positive change
- [ ] Ready to request testimonial

### Long-Term (1 Month)

- [ ] Transitioned to monthly check-in schedule
- [ ] Support requests < 2 hours total
- [ ] Senior reports improved connection with family
- [ ] Open to referral request

## Appendix: Scripts

### Introduction Script
```
"Hi, I'm [Name] from Elder Tech Concierge. I'm here to help set up your
new assistant. This is going to be really easy - we'll go at your pace,
and I'll show you just a few things today. The goal is to help you stay
connected with your family more easily. Do you have any questions before
we get started?"
```

### Training Completion Script
```
"You did great today! Remember, you can always just ask the Concierge for
help if you forget what to say. I'm going to call you in a couple of days
to check in and answer any questions. And your [daughter] knows how to
reach me if anything comes up. Is there anything else you'd like to
practice before I go?"
```

### Follow-Up Call Script (Day 2)
```
"Hi [Name], this is [Technician] from Elder Tech Concierge. I'm calling
to see how things are going with your new assistant. Have you had a
chance to try sending a text or checking the weather? ... Great! Do you
have any questions for me? ... Remember, you can call me anytime at
[phone]. Talk to you next week!"
```
