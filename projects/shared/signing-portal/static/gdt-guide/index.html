GD&T Symbols

Form Controls (No Datum Required)

Symbol Name What It Controls Tolerance Zone
⏤ Flatness How flat a surface is Two parallel planes
⌭ Straightness How straight a line/axis is Two parallel lines or cylinder
○ Circularity (Roundness) How round a cross-section is Two concentric circles
⌓ Cylindricity Combined roundness + straightness of cylinder Two concentric cylinders

Orientation Controls (Datum Required)

Symbol Name What It Controls Tolerance Zone
⟂ Perpendicularity 90° relationship to datum Two parallel planes/cylinder at 90° to datum
∠ Angularity Angle relationship to datum (not 90°) Two parallel planes at specified angle
∥ Parallelism Parallel relationship to datum Two parallel planes/cylinder parallel to datum

Location Controls (Datum Required)

Symbol Name What It Controls Tolerance Zone
⌖ Position Location of feature from datums Cylindrical zone (holes) or parallel planes
◎ Concentricity Center points alignment Cylindrical zone about datum axis
⌯ Symmetry Symmetrical relationship to datum Two parallel planes centered on datum

Profile Controls

Symbol Name What It Controls Datum Required?
⌒ Profile of a Line 2D profile shape at any cross-section Optional (controls form, orientation, location)
⌓ Profile of a Surface 3D surface shape Optional (controls form, orientation, location)

Runout Controls (Datum Required)

Symbol Name What It Controls Measurement
↗ Circular Runout Surface variation at any single cross-section during rotation FIM (Full Indicator Movement) at each circle
↗↗ Total Runout Surface variation across entire surface during rotation FIM across all circles simultaneously

  • First compartment
  • Identifies the geometric characteristic
  • One of the 14 GD&T symbols

  • Second compartment
  • ⌀ = diametral zone (for holes/pins)
  • Tolerance value in drawing units
  • Material modifiers (Ⓜ, Ⓛ, etc.)

  • Third compartment
  • Most important reference

  • Fourth & fifth compartments
  • Additional references
  • Order = constraint priority

Material Condition Modifiers

Ⓜ
MMC — Maximum Material Condition
Feature contains max material (smallest hole, largest pin). Provides BONUS tolerance as feature departs from MMC.
Ⓛ
LMC — Least Material Condition
Feature contains least material (largest hole, smallest pin). Bonus tolerance as feature departs from LMC.
Ⓢ
RFS — Regardless of Feature Size
Default condition. Tolerance applies at any size. No bonus tolerance. (Implied if no modifier shown)
Ⓟ
Projected Tolerance Zone
Tolerance zone projects beyond the feature (e.g., above a threaded hole for stud clearance).
Ⓕ
Free State
Tolerance applies in free state (unrestrained). Used for flexible parts.
Ⓣ
Tangent Plane
Tolerance applies to a plane tangent to the surface (not the surface itself).

💡 Bonus Tolerance (MMC/LMC)

Bonus = |Actual Size - MMC Size|
If a ⌀10.0 ±0.1 hole is at ⌀10.05 (actual) with position tolerance ⌀0.25 at MMC:
Bonus = |10.05 - 9.9| = 0.15 → Total allowed position = 0.25 + 0.15 = ⌀0.40

Datum Reference Frame

Datums establish the coordinate system for measurement. They constrain degrees of freedom in order: Primary → Secondary → Tertiary.

A
B
C

Datum letters in squares on drawings

Primary Datum (A)

  • Removes 3 DOF (plane) or 4 DOF (cylinder)
  • Part sits on this first
  • Usually the largest/most stable surface

Secondary Datum (B)

  • Removes 2 additional DOF
  • Perpendicular to primary
  • Orients the part

Tertiary Datum (C)

  • Removes final DOF
  • Fully constrains part
  • Locates in remaining direction

🔧 Datum Selection Tips

• Choose functional surfaces (mating faces, bearing surfaces)
• Primary should be large, accessible, stable
• Consider how the part will be fixtured in manufacturing
• Match datum structure to assembly sequence when possible

Fundamental Rules

Rule #1 — Envelope Rule (Taylor Principle)

The surface of a feature of size cannot extend beyond the MMC boundary (perfect form at MMC). Applies to individual features of size unless overridden by a straightness or flatness callout.

Rule #2 — Default RFS

Unless MMC Ⓜ or LMC Ⓛ is specified, RFS (Regardless of Feature Size) applies. No bonus tolerance is available by default.

Rule #3 — Screw Thread Convention

For screw threads, each tolerance and datum reference applies to the pitch cylinder (axis) unless otherwise specified.

Quick Formulas

📍 Position (Holes)

Position Error = 2 × √(X² + Y²)

Where X, Y = deviation from true position

🎯 Bonus Tolerance

Bonus = |Actual Size - MMC|

Total Tol = Stated Tol + Bonus

🔩 Fixed Fastener

T = H - F

T = position tolerance (each part)
H = MMC hole size
F = MMC fastener size

🔩 Floating Fastener

T = (H - F) / 2

Divide by 2 because tolerance
is shared between two parts

Common Mistakes to Avoid

❌ Using Position Without Datums

Position ALWAYS requires datum references. Without them, there's no origin to measure from.

❌ Circular Tolerance on Holes

Don't use ± for hole positions. Use ⌀ (diametral) position tolerance — it gives 57% more tolerance area.

❌ Over-Tolerancing

Only call out what matters functionally. More GD&T ≠ better parts. It = higher cost.

❌ Wrong Datum Order

Datum order matters. A|B|C ≠ B|A|C. Match your datum structure to functional requirements.
