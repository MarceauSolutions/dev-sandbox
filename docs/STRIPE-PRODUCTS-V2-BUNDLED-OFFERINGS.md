# Stripe Products V2 - Bundled Website + Automation Offerings

**Created:** 2026-01-19
**Status:** ACTIVE - Replace previous standalone automation tiers
**Rationale:** Most clients need website + automation together. Bundling increases value, simplifies sales, higher revenue per client.

---

## Product Strategy: Vertical Integration

**Problem Identified:**
- Many small businesses have no website (55% of scraped Naples businesses)
- Automation requires website (forms, booking, integrations)
- Selling automation without website creates friction: "I need to buy a website first?"
- Website without automation is just a brochure (no conversion)

**Solution:**
Bundle website + automation into single offering. Higher perceived value, single price, seamless integration.
