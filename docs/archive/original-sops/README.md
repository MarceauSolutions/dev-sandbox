## Purpose

This archive preserves the original SOP text for human reference. The active versions (which may have minor formatting updates) are in `docs/sops/`.

## Changes Made During Restructuring

The restructuring was a formatting/organization change only — no SOP content was modified:
- SOPs were extracted from the monolithic file into individual files
- Each SOP was given a consistent filename format
- An INDEX.md was created for navigation
- SOPs 25, 27, 28, 29 already existed as standalone files in `docs/`
