# Documentation Archive

This directory contains documentation that is no longer actively referenced but preserved for historical context.

## Structure

```
docs/archive/
├── session-files/     # Date-stamped session analysis files
├── analysis/          # One-time research and analysis documents
└── deprecated/        # Superseded documentation
```

## What Goes Where

| Category | Example Files | Criteria |
|----------|---------------|----------|
| **session-files/** | `2026-01-15-billing-analysis.md` | One-time session work, date-prefixed |
| **analysis/** | `GOOGLE-CLOUD-COST-ANALYSIS.md` | Deep-dive analysis, completed and closed |
| **deprecated/** | `FOLDER-STRUCTURE-GUIDE-OLD.md` | Replaced by newer version |

## When to Archive

Move files here when:
- Analysis is complete and no longer actively referenced
- File has been superseded by a newer version
- Session-specific work that served its purpose
- Documentation for deprecated features/systems

## Active Documentation

Keep in `docs/` (not archive) if:
- Actively referenced by CLAUDE.md or SOPs
- Living documentation that gets updated
- Core operational guides
- Quick-reference materials

## File Naming

- Session files: `YYYY-MM-DD-description.md`
- Analysis: Keep original name
- Deprecated: Keep original name (shows lineage)
