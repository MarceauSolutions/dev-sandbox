# Virtual Mailbox Setup Guide for Marceau Solutions

*Last Updated: 2026-01-27*
*Status: Setup guide*

## Overview

A virtual mailbox provides a professional business address without needing physical office space. Essential for:
- LLC/business registration (registered agent)
- Client correspondence
- Privacy (keep home address off public records)
- Professional appearance

## Setup Checklist

### Pre-Setup
- [ ] Decide on business address location (same state as LLC registration)
- [ ] Compare provider pricing for that location
- [ ] Check if location can serve as registered agent

### Account Setup
- [ ] Create account with chosen provider
- [ ] Select plan (Standard/Business tier recommended)
- [ ] Complete USPS Form 1583

### USPS Form 1583 (Required)
This form authorizes the mailbox service to receive mail on your behalf.

**Requirements**:
1. Fill out form (provider usually sends link)
2. Get notarized (UPS Store, bank, or mobile notary)
3. Upload/submit notarized form to provider
4. Wait for verification (1-3 business days)

**What you need for notarization**:
- Photo ID (driver's license or passport)
- Completed Form 1583
- ~$10-15 notary fee

### Post-Setup
- [ ] Test mail delivery (send yourself a test letter)
- [ ] Update LLC registered agent address (if using for this)
- [ ] Update business cards/website with new address
- [ ] Set up mail forwarding from old addresses (if applicable)
- [ ] Configure notification preferences (email/app alerts)

## Address Format

Once set up, your address will look like:
```
William Marceau
Marceau Solutions LLC
1234 Main Street #567
Naples, FL 34102
```

The "#567" is your personal mailbox number (sometimes "PMB 567" or "Suite 567").

## Integration with Business Registration

### Florida LLC
If using virtual mailbox as registered agent address:
1. Ensure provider allows registered agent use
2. Address must be in Florida
3. Update SunBiz.org with new address
4. Provider must accept legal documents on your behalf

### IRS/Tax Correspondence
- Virtual mailbox address can be used for IRS correspondence
- Make sure provider scans IRS mail promptly
- Consider premium tier for tax season

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Mail returned to sender | Verify USPS 1583 is processed; contact provider |
| Wrong items scanned | Contact support; request re-scan |
| Delayed notifications | Check spam folder; verify email in account |
